# Async/Await Practice – Per Scholas Lab

This project demonstrates:

- Custom API simulation with Promises
- Async/Await functions
- Retry mechanism for failed API calls
- Loading states
- Random error generation (like real APIs)
- Separation of files using modules

# How to Run
- npm install
- npm install typescript --save-dev
## Compile TypeScript to JavaScript
- npx tsc
- dist/
- node dist/index.

## Step              

-         `npm install`        Install dependencies       
-         `npx tsc`             Compile TypeScript         
-         `node dist/index.js`  Run the project            
-  `npm start`           Build + run in one command 



## Files

- **apiSimulator.ts** – Fake API functions
- **customerService.ts** – Async wrapper with retry logic
- **utils.ts** – Helper utilities
- **index.ts** – Main runner
